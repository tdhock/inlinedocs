** Installation

As of 3 June 2014, inlinedocs from R-Forge is deprecated (please do
not use it). To install the up-to-date inlinedocs from this GitHub
repository,

#+BEGIN_SRC R
install.packages("devtools")
devtools::install_github("tdhock/inlinedocs")
#+END_SRC
